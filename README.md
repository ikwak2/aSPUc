### aSPUc

Some C++ coded aSPU tests that do not pass the CRAN test (aSPUperm2C, aSPUsPath2C, aSPUpath2C and aSPUsPathC). They are all much faster than R coded version in aSPU package.

### installation
With `devtools`:
```S
library(devtools)
install_github("ikwak2/aSPU")
```
